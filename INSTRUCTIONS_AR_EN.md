SANITY CMS CONTENT GUIDE (NO CODING) - ENGLISH + ARABIC
=======================================================

ENGLISH (FOR NON-TECHNICAL USE)
--------------------------------

This guide is only for adding and editing content inside Sanity CMS.
No coding, no terminal, no installation.

1) WHERE TO EDIT CONTENT
- Open Sanity Studio: https://mohamedsalem.sanity.studio
- Log in with your account.
- Use the left sidebar to choose content type.

2) HOW TO SAVE CORRECTLY
- After filling fields, click Publish.
- If you only click Save, the content may stay as draft.
- Website updates may take up to 1 minute, then refresh the page.

3) CONTENT ORDER (IMPORTANT)
Add data in this order:
1. Site Settings
2. Categories
3. Projects
4. Testimonials
5. Services

4) SITE SETTINGS (ONE PAGE ONLY)
Purpose: main personal info used across Home, About, Contact.

Fill these fields:
- Name: Mohamed Salem
- Tagline: short professional line
- Short Bio: 2-4 lines about you
- About Text: full paragraph(s) for About page
- Portrait: your main photo
- Availability Status: Available / Limited / Not Available
- Email: your contact email
- Phone: optional
- Location: e.g. Giza, Egypt
- Social Links: add platform + URL for each social account
- Behance URL: your Behance link
- Resume File: optional PDF

5) CATEGORIES
Purpose: organize and filter projects.

For each category fill:
- Title (required)
- Slug (generate from title)
- Description (optional)

Recommended categories:
- Brand Identity
- Logo Design
- Packaging
- Illustration
- Social Media
- Print

6) PROJECTS
Purpose: portfolio items.

For each project fill:
- Title (required)
- Slug (required)
- Client (optional)
- Category (select one)
- Year
- Thumbnail (required)
- Hero Image (recommended)
- Body (project story and details)
- Gallery (extra images)
- Featured (turn ON if you want it on Home page)
- Order (1,2,3... for sorting)

Important:
- Home page shows only projects where Featured is ON.
- Publish each project after editing.

7) TESTIMONIALS
Purpose: client reviews.

For each testimonial fill:
- Client Name (required)
- Role (optional)
- Company (optional)
- Quote (required)
- Avatar (optional)
- Featured (turn ON to show on Home page)

Important:
- Home page shows only testimonials where Featured is ON.

8) SERVICES
Purpose: service cards on Services page.

For each service fill:
- Title (required)
- Description
- Features (list items)
- Icon Name (optional)
- Order (for sorting)
- Pricing Label (example: Get a Quote)

Supported icon names:
- palette
- pen-tool
- package
- brush
- share2
- file-text

9) QUICK CHECK IF CONTENT DOES NOT APPEAR
- Is it Published?
- Is Featured ON? (for Home projects/testimonials)
- Wait 60 seconds and refresh.
- Check you edited the correct document.


العربية (للاستخدام بدون خبرة برمجية)
-------------------------------------

هذا الدليل فقط لإدخال وتعديل المحتوى داخل Sanity CMS.
لا يحتاج كود، ولا أوامر، ولا تثبيت.

1) أين تعدل المحتوى
- افتح لوحة Sanity: https://mohamedsalem.sanity.studio
- سجّل الدخول بحسابك.
- من القائمة اليسار اختر نوع المحتوى.

2) طريقة الحفظ الصحيحة
- بعد إدخال البيانات اضغط Publish.
- إذا ضغطت Save فقط قد يبقى المحتوى Draft.
- تحديث الموقع قد يأخذ حتى دقيقة ثم اعمل Refresh.

3) ترتيب إدخال المحتوى (مهم)
أدخل البيانات بهذا الترتيب:
1. Site Settings
2. Categories
3. Projects
4. Testimonials
5. Services

4) Site Settings (صفحة واحدة فقط)
الغرض: البيانات الأساسية التي تظهر في Home وAbout وContact.

املأ الحقول التالية:
- Name: Mohamed Salem
- Tagline: جملة تعريفية قصيرة
- Short Bio: نبذة قصيرة 2-4 سطور
- About Text: نص كامل لصفحة About
- Portrait: الصورة الشخصية الأساسية
- Availability Status: متاح / متاح جزئيا / غير متاح
- Email: البريد للتواصل
- Phone: اختياري
- Location: مثال Giza, Egypt
- Social Links: أضف المنصة + الرابط لكل حساب
- Behance URL: رابط Behance
- Resume File: ملف PDF اختياري

5) Categories
الغرض: تنظيم المشاريع وعمل فلترة.

لكل قسم املأ:
- Title (إجباري)
- Slug (أنشئه من Title)
- Description (اختياري)

أقسام مقترحة:
- Brand Identity
- Logo Design
- Packaging
- Illustration
- Social Media
- Print

6) Projects
الغرض: عرض أعمال البورتفوليو.

لكل مشروع املأ:
- Title (إجباري)
- Slug (إجباري)
- Client (اختياري)
- Category (اختَر قسم)
- Year
- Thumbnail (إجباري)
- Hero Image (مستحسن)
- Body (شرح المشروع والتفاصيل)
- Gallery (صور إضافية)
- Featured (فعّلها إذا تريد ظهوره في الصفحة الرئيسية)
- Order (1,2,3... للترتيب)

مهم:
- الصفحة الرئيسية تعرض فقط المشاريع التي Featured فيها ON.

7) Testimonials
الغرض: عرض آراء العملاء.

لكل رأي عميل املأ:
- Client Name (إجباري)
- Role (اختياري)
- Company (اختياري)
- Quote (إجباري)
- Avatar (اختياري)
- Featured (فعّلها ليظهر في الصفحة الرئيسية)

مهم:
- الصفحة الرئيسية تعرض فقط Testimonials التي Featured فيها ON.

8) Services
الغرض: بطاقات الخدمات في صفحة الخدمات.

لكل خدمة املأ:
- Title (إجباري)
- Description
- Features (قائمة نقاط)
- Icon Name (اختياري)
- Order (للترتيب)
- Pricing Label (مثال: Get a Quote)

أسماء أيقونات مدعومة:
- palette
- pen-tool
- package
- brush
- share2
- file-text

9) فحص سريع إذا المحتوى لا يظهر
- هل ضغطت Publish؟
- هل Featured مفعلة؟ (للمشاريع والآراء في الصفحة الرئيسية)
- انتظر 60 ثانية ثم Refresh.
- تأكد أنك عدلت المستند الصحيح.
